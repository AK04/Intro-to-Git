Connecting with repositories

Adding urls

You can sync with repositories uploaded online, by adding connections urls using git remote. The urls will be seen on the repositories webpage. With these urls, you can download and upload your updated versions(we will get to that shortly) of the repository. To add the url, use this command:
git remote add name url
You can then refer to this url by the name you gave when you need to download or upload the files. For example, you can add my repository's url like this:
git remote albin https://github.com/AK04/Guess-game.git

Checking, Renaming and Deleting urls

You can see all the urls you have added using this command:
git remote -v

You can rename your url using this command:
git remote rename oldname newname

To delete your url:
git remote rm name
